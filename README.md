## 🛠️ Cara Menjalankan Project (Local Environment)

<details>
<summary><b>Klik di sini untuk melihat instruksi instalasi</b></summary>
<br>

1. **Clone repository ini ke komputer lokal** <br>
   Buka terminal atau Git Bash, lalu jalankan perintah berikut:
   ```bash
   git clone https://github.com/mnurkholish/242410101063-PWEB.git
   ```
2. **Prosedur Menjalankan Program:**

- **Bagi Tugas HTML, CSS, & Javascript:**
  File utama (misalnya: <small>`index.html`</small>) dapat dijalankan secara langsung melalui peramban (_browser_). Penggunaan ekstensi **Live Server** pada Visual Studio Code juga sangat direkomendasikan. <br>
  Eksekusi juga dapat dilakukan melalui terminal dengan perintah berikut, bergantung pada sistem operasi yang digunakan:

  **Windows:**

  ```Bash
  start index.html
  ```

  **macOS:**

  ```Bash
  open index.html
  ```

  **Linux:**

  ```Bash
  xdg-open index.html
  ```
